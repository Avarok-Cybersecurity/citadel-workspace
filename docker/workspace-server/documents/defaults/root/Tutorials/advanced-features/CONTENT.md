# Advanced Features

Take your Citadel experience to the next level.

## Live Documents

Create collaborative documents that sync in real-time:
1. Click the "Create Live Document" button in chat
2. Share the document link with collaborators
3. Edit together with instant synchronization

## P2P Messaging

Send direct messages to other users:
1. Click a user's name to start a conversation
2. Messages are end-to-end encrypted
3. Share files and live documents

## Workspace Switching

Connected to multiple workspaces? Switch instantly:
1. Click your workspace icon in the sidebar
2. Select another workspace from the dropdown
3. Your session remains active in all workspaces

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + K` | Quick search |
| `Ctrl/Cmd + Enter` | Send message |
| `Escape` | Close modal |
