# Welcome to Citadel Workspaces

**Citadel Workspaces** is a hyper-secure, decentralized collaboration platform designed for teams who value privacy, data sovereignty, simplicity, and real-time workplace synchronicity for managing remote work and projects.

## Getting Started

1. **Explore Offices** - Use the sidebar to navigate between different offices. Inspired by natural design, each *office* has its own set of *rooms*, just like a physical workplace.
2. **Join the Discussion** - Click the Chat tab to participate in conversations per office, or per room, or between others.
3. **Check Announcements** - Stay up to date with the latest news
